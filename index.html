What is Droix?

Droix is an Android repair framework that automatically fixes crashes in Android apps. Given a buggy APK and an UI event sequence that crashes on the buggy APK, Droix uses a set of lifecycle-aware operators to produce a fixed APK. Figure below shows Droix's workflow
